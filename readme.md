## Implementation

Esta app ha sido creada utilizando Flask.

# Initialized the database.

flask --app flaskr init-db

# Run the app:
flask --app flaskr run --debug

# Run all Tests
pytest

# Coverage
coverage run -m pytest

# Coverage report

coverage report

# Export coverage report in html

coverage html
